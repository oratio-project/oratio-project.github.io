ERP

http://it.wikipedia.org/wiki/Enterprise_resource_planning

ERP è un sistema di gestione, chiamato in informatica sistema informativo, che integra tutti i processi di business rilevanti di un'azienda:

  • Gestione di produzione
  • Marketing e vendite
  • Gestione magazzino
  • Contabilità e analisi finanziaria

Tecnologia basata sul web

  • Linguaggio di programmazione Perl
  • Java
  • PostgreSQL
  • Firefox Mozilla
